How Groundwork works

1

Capture

Scans your repo on install (CLAUDE.md, package.json, Prisma schema) and extracts decisions from live AI sessions.

2

Propagate

Injects the right decisions into every teammate's AI context before they write a line — via the Model Context Protocol.

3

Detect

Flags conflicts in real time. A UUID decision and an incoming integer-ID plan get caught before code is written.

4

Enforce

A GitHub Action checks every PR against the decision graph and blocks merges that violate a P0 decision.
